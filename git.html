Git and GitHub Tutorial






Intro

When making digital projects, espicaly with other people, you should have a way to easly remotly store the projects and see the version history. Git is an open source software that allows for the this, and stores each project to "repositories". If you when git to store your project remotly, you need a git server. You can host your own git server or use a public one. Github is the most popular public git service.



Github Setup

Go to github.com and sign up. Click you profile > Go to settings > Click SSH and GPG keys > Click new SSH key.


In terminal, paste this with your email:

ssh-keygen -t ed25519 -C "your_email@example.com"
cat ~/.ssh/id_ed25519.pub

You can put in all the defults by clicking enter.
Paste the key into add new and add a title. Finaly, click add key.

This will allow you to change github repos on tied with your github account on the computer the ssh is associated with. To change repos for koibots, you will need to joun the github organization. Ask to be invited, and accept the invintation to koibots. Now you can edit the repos.
