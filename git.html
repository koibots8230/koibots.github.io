Git and GitHub Tutorial

Intro

When making digital projects, especially with other people, you should have a way to easily remotely store the projects and see the version history. Git is an open source software that allows for this. It stores each project in a repository (repo). If you when git to store your project remotely, you need a git server. You can host your own git server or use a public one. GitHub is the most popular public git service.

Github Setup

SSH Keys

Go to github.com and sign up. Click you profile > Go to settings > Click SSH and GPG keys > Click new SSH key.

In terminal, paste this with your email:

ssh-keygen -t ed25519 -C "your_email@example.com"
cat ~/.ssh/id_ed25519.pub

You can put in all the defaults by clicking enter. Paste the key into add new and add a title. Finally, click add key.

This will allow you to change GitHub repos on tied with your GitHub account on the computer the ssh is associated with. To change repos for KoiBots, you will need to join the GitHub organization. Ask to be invited, and accept the invitation to KoiBots. Now you can edit the repos.

Cloning Repos

To be able to edit a repo, you have to clone it first. To clone a repo use the following command:

git clone "git@github.com:user-name/repo-name.git"

To clone the repo this website is in use:

git clone "git@github.com:koibots8230/koidocs.git"
