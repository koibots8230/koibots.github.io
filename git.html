Git and GitHub Tutorial






Intro

When making digital projects, espicaly with other people, you should have a way to easly remotly store the projects and see the version history. Git is an open source software that allows for the this. If you when git to store your project remotly, you need a git server. You can host your own git server or use a public one. Github is the most popular public git service.
